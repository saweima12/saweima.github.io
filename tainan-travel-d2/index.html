0725 台南四日遊 - 悠閒的美食之旅 Day2

早上睜開眼睛，發現自己見到的不是看慣了的天花板，才真正有了在外旅行的實感。由於第一天忙著整理行李及安置什麼都沒逛到，因此第二天可以說是台南探索之旅的正式開始。由於飯店有提供簡單的早餐，所以早餐的開銷是直接省了不少（可惜我忘記拍照）提供的是簡單的麵包、牛奶、紅茶、咖啡的自助吧，可以說是非常的全面，以當初下訂不到 500 元的價格，真的是值回票價。

由於是初次旅行，對行程安排不大熟悉，搞的有些彆扭。今天的預定是要前往安平區逛逛老街跟古堡，但是安平區沒能訂到便宜的青年旅館，所以才選擇居住在巷弄輕旅，但退房後隨即發現自己身上背著一大包行李根本不好行動（然後我壓根忘記了有行李寄放這服務）。

因此只能先前往第二天要居住的旅館希望可以先放置行李，這也導致我第一天住在巷弄輕旅這就變得完全沒有意義了。

# 前往第二天的旅店

由於距離第二天居住的旅店有些距離，加上天氣有夠熱，太陽彷彿能曬死人似的，因此決定尋找附近的交通工具。在台南也有類似台北uBike 的公共腳踏車 t-bike ，嗯...依然有些距離。

第二天居住的旅館一樣是在 Agoda 預定的青年旅館（偉哉 Agoda ) 資料如下：

名稱：快活慢行
網址：Official Website

一樓就是咖啡廳，非常的漂亮。走進門就能夠聞到一股濃厚的咖啡香，西式的咖啡吧台配上塌塌米也不顯的突兀，顯的非常有設計感。房屋的外觀上可以看出是有些年紀的房子改建成的民宿。 星期六的時候櫃台只服務到下午四點算是美中不足，所以 Check In 的時間要拿捏好。

由於人生地不熟，加上第一次旅行，我想都沒想上午 10 點就直接先去 Check In 了，完全忘了還有整理房間的這段時間。本來打算先寄放行李不過基於自己多疑的性格，不太敢隨便把行李寄放，因此最後還是決定背在自己身上。這個決定讓我在後面吃了不少苦頭。

# 前往安平老街及出張所

魯莽的決定所帶來的絕對不是好運。由於帶著行李又沒租機車，加上自己的固執（不想租！），打算只靠 T-Bike 跟 走路逛完這四天，因此也吃了不少苦頭。 由於沒有做好功課，我沒發現 T-Bike 有安億停車場站，因此只騎到民生轉運站就還車用走的，殊不知後面還有一大段路要走。

第一目標是安平老街、接下來一路往出張所走，路上也經過了不少店面、看了不少風景。這時也剛好快要接近正午，太陽可以說是非常的大。由於口乾舌燥、加上背著近三公斤的行李行走（有夠自虐啊）雖然有攜帶水壺但裡面只有從巷弄輕旅裝到的溫水。 這時正好路過了樁之味。

名稱：椿之味薏仁湯
網址： Map Position

由於當時接近正午，太陽非常的大，想要稍作休息、順便點杯飲料來喝。剛好看到了菜單上的紅豆鮮奶薏仁，就立刻點了一杯，結果原來是用湯匙吃的。（圖片由於是邊走邊吃手拿的有點不太穩，而且等我想起要拍照時已經吃一半了）

濃厚的牛奶、滿滿的薏仁味再加上我最愛的紅豆吃起來冰冰涼涼的口感非常的消暑，非常的好吃。（雖然朋友表示不點綠豆根本邪教）

走著走著終於來到了安平老街，高中時曾在這攤吃過一次，算是自己對台南為數不多的記憶點之一。逛著逛著也感嘆老街中的品項也跟其他夜市的內容相差無幾，一成不變的，有些缺乏特色。記憶點回顧完後，其實也不知道該去哪了，只能在 Google Map 上隨意的找些附近的景點走走。

路過了炸雞洋行，本來看名稱以為是德記洋行之類的舊設施改建成的炸雞攤位覺得很有特色，不過晚些回程時才發現是連鎖店，有些小失望。 由於接近正午肚子也有些餓，因此買了份炸雞跟薯條後繼續趕路。

最後終於來到了夕遊-出張所，專門賣鹽的奇妙地方，另外內有販售鹽味冰淇淋，吃起來還蠻有特色的。

# 觀汐平台與偷懶的回程

在出張所稍做歇息後，突然覺得既然都來到台南了，怎麼能不看一下海呢？ 便動身想前往最近的海邊。 本來看著 Google 地圖，想直接前往出張所北邊的湖濱水鳥公園看看，不過由於季節不對，沒看到任何鳥。（失望）

接著便朝著觀汐平台出發，很棒的是貌似正好遇上浪人祭的主辦日子，在安平港遠遠的就能聽到演唱會的歌聲算是賺到了，同時也因為演唱會的關係，周遭有著許許多多的攤販，在這夏日炎炎的正午烈日，能夠方便的補充水分也算是種幸運。

在觀汐平台的附近歇息著，吹著海風、曬著太陽，讓大腦靜靜的放空，享受著片刻的寧靜。也算是睽違好幾年再度見到大海，覺得一切都算是值得了。約休息了半小時，才準備離去。 走到一半才發現有公車可以直接通到市區，忽然覺得自己為何要走那麼多路呢？超愚蠢的....

# 五黑家日式拉麵及武聖夜市

回到旅店後，由於大太陽之下又背著數公斤的行李行走了很長一段時間，所以稍作盥洗並休息，等待著夜晚與當地朋友的聚餐。

名稱： 五黑家日式拉麵 網址： Map Position

屬於可選麵條硬度、湯頭濃度的雞白湯拉麵，味道濃郁且麵條 Q 彈，非常的推薦。來本來想要點些炸雞作為小菜，可惜的是都已經賣完了，算是有些許的遺憾。

其實本來打算聚餐後發完雞排就直接回到旅店休息的，不過由於當地朋友一句「沒來過武聖夜市，別說你來過台南。」的驅使下，還是決定去逛逛武聖夜市。

算是第一次見到有射箭攤位的夜市，老闆跟老闆娘會很熱心的幫你橋姿勢直到你能安穩把十隻箭射出去為止。很遺憾我的臂力不夠大，沒辦法拉滿弦就是。

再來就是排隊購買號稱是武聖夜市的特色美食「沙威瑪」，今天來的時候總共見到了三個攤位在賣沙威瑪，不過兩邊排隊的人潮差的非常的多，基於好奇心，因此兩邊的都買了一份來研究。

總體而言，差別最大的大概就是配料了，兩邊的肉吃起來其實差異並不大，但是在麵包、蔬菜、醬料的配置上則相差不少，排隊那邊的沙威瑪的麵包雖然烤過但吃起來依然非常的鬆軟、美乃滋塗的也只有薄薄的一層、醬料也加的很少，可以完全的吃出沙威瑪的肉、蔬菜、麵包的美味以及體會到那咬下的層次感。

另一邊的就比較不優了些，麵包烤過後表皮變得脆脆的勉強算是一種特色，不過美乃滋及各種醬料就變成了各種加好加滿，生菜的數量也意外的有點少，吃起來就沒有前者的變化那麼多了。

吃完後，由於今日走了許多路，有些勞累便早早歇息了。

# 本日小結

本來的初衷是希望接下來的三天都能逛逛不同的地區，所以三個晚上都居住在不同的旅店。但在實際進行過一次之後就覺得這根本是累死自己的舉動。 由於每間旅館的退房時間、整理時間不一，所以不大好掌握入住的時間。 也沒注意好夜市的開啟時間，不然第一天跟第二天的旅館位置應該互換才對。

不過整體而言今日的體驗並不差，嘗試了射箭、看到了大海、吃了排隊美食，基本上打算來台南的目標也算是達成了一半了。
