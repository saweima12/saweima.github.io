0724 台南四日遊 - 悠閒的美食之旅 Day1

年初時，因為一些原因而發了篇雞排的祭品文，由於其中的參與者遍布台灣各地，加上雞排這種東西當面發感覺比較有誠意（？）同時也能順便當網聚，因此就決定找些時間四處走走順便發雞排...

可惜，由於春節後武漢肺炎疫情的突然爆發，所以一時之間沒有了下文，一直到六月開始逐漸解禁後，才再次計畫了這次的旅遊。

這也是我自己第一次獨自旅行，心裡多少有些怕怕的。訂房、車票、路線規劃等都由自己一手包辦，處理起來手忙腳亂，加上正逢暑假又是振興旅遊的時期，因此飯店也並不是很好訂，弄著弄著八小時就過去了，只能說也算是種經驗。

# 出火車站、尋找旅館

剛踏出火車站，面對艷陽高照的台南。對於一個經常待在冷氣房的工程師來說還挺殺人的。出來後的第一件事情便是查地圖尋找當天的旅館在哪，畢竟身上滿是行李也挺重的。

第一晚的住宿旅店是在 Agoda 訂房網站上訂的巷弄輕旅，資料如下：

名稱： 巷弄輕旅 Oinn Hotel & Hostel
網址： Official Website

這也是首次嘗試住在青年旅館，一來節省旅費、二來認為也是不錯的體驗。每個人都有自己的獨立床位及保險櫃（雖然這在青年旅館算是標配了），比較特別的是每個床位都有裝設電視，晚上可以躺在床上觀看。

另外也有提供早餐，除了交通位置不太好之外，CP值跟獲得的體驗都非常不錯，推薦一試。

# 尋找週邊的晚餐

由於對周遭的食物並不熟，因此求助於住在台南的網友們推薦。最後因為距離剛好挺近的所以選擇了這間。

名稱： 阿興虱目魚
網址： Map Position

雖然招牌上寫著虱目魚，友人卻是力推蒜頭飯跟蒜泥白肉（也有價格上的考量），味道上是非常不錯，不管是飯、是肉、是菜吃起來都非常的香（而且還有點甜甜的）。

吃完之後，看了下地圖發現旅館附近就有武聖夜市，不過走過去才發現由於今天是星期五所以沒開，有些失望，加上突然開始懷念小時候吃的鱔魚麵，因此特別繞了一段路去尋找好吃的鱔魚麵。

名稱： 德興鱔魚意麵
網址： Map Position

由於是晚餐時段，所以稍微等了一陣子，放入口中那有多種層次的味道讓我非常的流連忘返，雖然相對於我印象中的味道還有些差距，但真的美味。推薦有到台南的人都可以嘗試一次。

# 今日總結

由於抵達台南時已經下午四點多，加上走路抵達旅館並整理好行李跟稍作休息就已經接近下午六點了，加上夜市沒開，所以還沒開始探索。只對周遭的食物進行嘗試，蒜頭飯跟鱔魚麵的味道真的讓我非常的驚豔，若有機會到台南的話真的非常推薦一試。
