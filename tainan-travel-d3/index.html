0726 台南四日遊 - 悠閒的美食之旅 Day3

轉眼間便來到了第三天，一早醒來下床便突然發現自己的腳長了水泡，果然人不做死就不會死。太過堅持走路，導致自己的鴨子腳撐不住了。有些行動受限，因此今天多以 T-Bike 作為代步工具。

幸好，想逛的地方在第二天都已經差不多逛完了，接下來只要把自己想吃的、想尋找的食物都走過一圈就行了。

# 快活慢行 Hiihubs - 早餐

名稱：快活慢行
網址：Official Website

第二晚的旅店也有提供早餐，雖然不像巷弄輕旅那樣自助取用吃到飽，但份量也絕對足夠，以簡餐式拼盤配上優格及飲料（能選擇豆漿、咖啡或紅茶），整體來說非常的營養均衡。

而令我最印象深刻的便是咖啡了，快活慢行的一樓平常就是咖啡廳，而他的咖啡也是非常的有特色。

與研磨咖啡機非常不同，聞起來淡淡的清香配上那不會太濃厚或太苦的味道再加上滑順、細膩的口感，是我從來沒喝過的味道，甚至回到了台北後的今天都回味無窮，若有機會的話非常推薦品嚐一次！

# 銀波布丁

在第二晚與友人們聚餐、逛夜市時，正好提到了布丁的相關事情，順便詢問了下周遭是否有推薦的甜食，便被推薦了銀波布丁。 仔細想想第二天走向安平時也經過了三間布丁店（然而忙著走路沒有進去，現在想想真是有點後悔，真該每間去吃一遍。）

名稱：銀波布丁

網址：Map Position

外表看上去感覺像是外面賣的純手工布丁一樣，放入嘴中品嚐時，綿密蓬鬆的口感配上甜甜的糖漿，有種另類的風味。

雖然在吃過得雞蛋布丁中不能排入前三名，不過或許只是我點錯了品項而已。而另一邊，個人比較喜歡的還是布丁冰沙，擁有雞蛋布丁的所有優點，加上當時台南的殺人太陽，簡直是消暑聖品。

# 康樂街牛肉湯

名稱： 康樂街牛肉湯
網址： Map Position

牛肉湯算是被列入前往台南時必吃的項目，雖然大多數人都說是早餐，不過我逛到這邊時已經是正午了。

不過由於這天剛好是假日，遊客的數量並不少，牛肉湯排了一個小時的隊才吃到，不過為了美食一切都值得。

在老闆娘的強烈推薦下嘗試了從沒吃過得品項 - 炒牛骨隨，算是從小到大都沒有吃過得東西，因此非常的好奇。 雖然剛拿到時還覺得挺神奇的，不過那放入口中入口即化且綿密的口感加上濃厚的牛肉香味，實在是讓人欲罷不能。

隨後送上的牛肉湯，有著蔬果熬煮的甜味，湯中的牛肉也是非常的鮮美，搭配著炒牛骨隨那微辣但綿密的口感，每一口都能獲得不同的感受。 如果沒有吃過牛骨隨的話非常推薦嘗試一次。

# 本日小結

比較可惜的是，今天的主要行程都花費在排隊上，老實說有些無所適從。不知道該去逛些什麼，只能把國華街大概有興趣的食物都吃過一遍。不過每次都吃完之後才發現忘記拍照，導致也沒什麼可講。

或許比起旅遊的路徑及過程，應該更注重地點的特色？ 在編寫上面還有些無所適從需要些方向。
